┌───────────────────────────────────────────────────────────────────────────┐
│ DATA FORMAT SPECIFICATION                                                 │
├───────────────────────────────────────────────────────────────────────────┤
│                                                                           │
│ Serial Data Format:                                                       │
│   DTA;31422;182;263;0;793;2238;0;611;0;!                                 │
│                                                                           │
│ Fields:                                                                   │
│   1.  Status (DTA/END)                                                    │
│   2.  Cycles (integer)                                                    │
│   3.  Position 1 (int ÷ 100 = mm)                                         │
│   4.  Force Lower (int ÷ 10 = N)                                          │
│   5.  Travel 1 (int ÷ 100 = mm, can be negative)                          │
│   6.  Position 2 (int ÷ 100 = mm)                                         │
│   7.  Force Upper (int ÷ 10 = N)                                          │
│   8.  Travel 2 (int ÷ 100 = mm, can be negative)                          │
│   9.  Travel at Upper (int ÷ 100 = mm)                                    │
│   10. Error Code (integer 0-999)                                          │
│   11. End Marker (!)                                                      │
│                                                                           │
│ Loss of Stiffness Calculation:                                            │
│   Formula: (Travel 2 / Travel at Upper) × 100%                            │
│   If Travel at Upper = 0, return 0%                                       │
└───────────────────────────────────────────────────────────────────────────┘
